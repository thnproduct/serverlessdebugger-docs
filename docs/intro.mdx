# Introduction

Start online debug sessions while the AWS Lambda functions run on AWS Cloud. 
Thundra sets up an easy and secure connection between your AWS Lambda functions and 
your favorite <a href="#getting-started">IDE</a>. Debug your serverless applications natively with their own permissions. 
Stop mocking resources and simulating the environments! Put breakpoints and pause
the execution of the <a href="/nodejs">Node.js</a>, <a href="/python">Python</a>  and <a href="/java">Java</a> functions to see the error, variables, stack trace and more!
